Programación Funcional¶

La programación funcional es un paradigma de programación en el que el flujo de ejecución del programa se basa en la evaluación de funciones. Este enfoque se basa en el cálculo lambda, un sistema formal que describe la computación en términos de funciones matemáticas.

Palabras clave y su definición¶

  • Paradigma de programación: Una forma de pensar sobre cómo resolver problemas de programación.
  • Evaluación de funciones: El proceso de aplicar una función a sus argumentos y obtener un resultado.
  • Cálculo lambda: Un sistema formal que describe la computación en términos de funciones matemáticas.
  • Inmutabilidad: Un concepto en programación funcional en el que los datos no pueden ser modificados una vez que se han creado.
  • Funciones de orden superior: Funciones que pueden tomar otras funciones como argumentos o devolver una función como resultado.
  • Recursión: Un método de resolución de problemas en el que una función se llama a sí misma para resolver un subproblema más pequeño.
  • Tipado estático: Un sistema en el que los tipos de datos se verifican en tiempo de compilación en lugar de en tiempo de ejecución.
  • Currying: Un proceso en el que una función con múltiples argumentos se convierte en una serie de funciones con un solo argumento.
  • Composición de funciones: La combinación de dos o más funciones para formar una nueva función.

Ejemplos de código en Haskell¶

Ejemplo 1: Función de suma¶

La siguiente función en Haskell recibe dos números enteros como argumentos y devuelve su suma:

suma :: Int -> Int -> Int
suma x y = x + y

Ejemplo 2: Función de orden superior¶

La función map es una función de orden superior en Haskell que toma una función y una lista como argumentos y devuelve una nueva lista aplicando la función a cada elemento de la lista:

map :: (a -> b) -> [a] -> [b]
map f [] = []
map f (x:xs) = f x : map f xs

Ejemplo 3: Función recursiva¶

La siguiente función en Haskell calcula el factorial de un número utilizando recursión:

factorial :: Int -> Int
factorial 0 = 1
factorial n = n * factorial (n - 1)

Ejercicios prácticos¶

  1. Crea una función en Haskell que calcule el área de un círculo dado su radio.
  2. Utiliza la función map para convertir una lista de números enteros en una lista de sus valores absolutos.
  3. Escribe una función recursiva en Haskell que calcule la potencia de un número dado su exponente.
  4. Crea una función de orden superior en Haskell que filtre los elementos de una lista según un criterio dado.
  5. Implementa una función en Haskell que calcule el promedio de una lista de números.

Consejos o mejores prácticas¶

  • Utiliza funciones puras en lugar de funciones con efectos laterales para evitar efectos secundarios no deseados.
  • Utiliza la recursión en lugar de bucles para resolver problemas en la programación funcional.
  • Aprovecha las funciones de orden superior para escribir código más conciso y legible.
  • Aprende a usar tipos de datos algebraicos para modelar problemas de manera elegante.
  • Practica el uso de funciones de composición para crear funciones más complejas a partir de funciones más simples.
