Funciones y Expresiones¶

Introducción¶

En Haskell, las funciones y expresiones son elementos fundamentales en la programación funcional. Una función es un bloque de código que toma uno o más valores de entrada, realiza una operación y devuelve un resultado. Una expresión, por otro lado, es una combinación de valores, operadores y funciones que se evalúa para obtener un resultado. En este módulo, aprenderás cómo utilizar funciones y expresiones en Haskell para resolver problemas de manera eficiente.

Palabras clave y Definiciones¶

  • Función: Una función es un bloque de código que toma uno o más valores de entrada, realiza una operación y devuelve un resultado.
  • Expresión: Una expresión es una combinación de valores, operadores y funciones que se evalúa para obtener un resultado.
  • Parámetro: Un parámetro es un valor que se pasa a una función para que pueda ser utilizado en su operación.
  • Argumento: Un argumento es un valor específico que se pasa a una función cuando se llama, para ser utilizado como un parámetro.
  • Tipo de datos: Un tipo de datos es una categoría para clasificar los valores en Haskell, como enteros, cadenas o booleanos.
  • Funciones de orden superior: Son funciones que toman otras funciones como argumentos o devuelven una función como resultado.
  • Composición de funciones: Es la combinación de dos o más funciones para formar una nueva función.

Ejemplos de Código en Haskell¶

Definición de una función¶

-- Función que suma dos números
suma :: Int -> Int -> Int
suma x y = x + y

Llamada a una función¶

-- Llamada a la función suma con 2 y 3 como argumentos
suma 2 3
-- Resultado: 5

Definición de una expresión¶

-- Expresión que calcula el área de un círculo
area :: Float -> Float
area r = pi * r^2

Llamada a una expresión¶

-- Llamada a la expresión area con 5 como argumento
area 5
-- Resultado: 78.53982

Funciones de Orden Superior¶

-- Función que aplica una función a cada elemento de una lista
map :: (a -> b) -> [a] -> [b]
map f [] = []
map f (x:xs) = f x : map f xs

Ejercicios Prácticos¶

  1. Define una función que calcule el área de un triángulo.
  2. Crea una expresión que convierta una temperatura en grados Celsius a grados Fahrenheit.
  3. Utiliza la función map para incrementar en 1 cada elemento de una lista de números.
  4. Crea una función que calcule el factorial de un número.
  5. Utiliza la composición de funciones para crear una función que calcule el doble del cuadrado de un número.

Consejos y Mejores Prácticas¶

  • Utiliza nombres descriptivos para tus funciones y expresiones.
  • Evita crear funciones y expresiones demasiado largas o complejas, es mejor dividirlas en funciones más pequeñas y reutilizables.
  • Aprovecha las funciones de orden superior para escribir código más conciso y elegante.
  • Utiliza la composición de funciones en lugar de anidar múltiples funciones en una sola expresión.
  • Prueba tus funciones y expresiones con distintos valores de entrada para asegurarte de que funcionan correctamente.
