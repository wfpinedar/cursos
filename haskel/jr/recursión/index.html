Recursión en Haskell¶

La recursión es una técnica utilizada en programación que consiste en la llamada a una función desde su propia definición. En Haskell, al ser un lenguaje funcional, la recursión es una herramienta fundamental para resolver problemas más complejos. En esta lección, aprenderemos cómo aplicar la recursión en Haskell y cómo puede ayudarnos a escribir código más eficiente y elegante.

Palabras clave y definiciones¶

  • Recursión: técnica utilizada en programación en la que una función se llama a sí misma desde su propia definición.
  • Función recursiva: función que se llama a sí misma dentro de su propio cuerpo.
  • Caso base: condición que detiene la recursión y evita que la función se llame infinitamente.
  • Pila de llamadas: estructura de datos que almacena las llamadas a funciones en orden de ejecución.
  • Funciones de orden superior: funciones que toman otras funciones como argumentos o devuelven funciones como resultado.

Ejemplos de código¶

Factorial¶

El factorial de un número es la multiplicación de todos los números enteros positivos desde 1 hasta el número en sí. Por ejemplo, el factorial de 5 es 5 * 4 * 3 * 2 * 1 = 120.

En Haskell, podemos escribir una función recursiva para calcular el factorial de un número de la siguiente manera:

factorial :: Int -> Int
factorial 0 = 1 -- caso base
factorial n = n * factorial (n-1) -- llamada recursiva

La función toma un número entero como argumento y devuelve su factorial. El caso base es cuando el número es 0, ya que el factorial de 0 es 1. La llamada recursiva se realiza restando 1 al número y multiplicando el resultado con la llamada recursiva de ese número.

Fibonacci¶

La secuencia de Fibonacci es una serie numérica en la que cada número es igual a la suma de los dos números anteriores. Por ejemplo, la secuencia comienza con 0, 1, 1, 2, 3, 5, 8, 13, ...

En Haskell, podemos escribir una función recursiva para generar la secuencia de Fibonacci de la siguiente manera:

fibonacci :: Int -> Int
fibonacci 0 = 0 -- caso base
fibonacci 1 = 1 -- caso base
fibonacci n = fibonacci (n-1) + fibonacci (n-2) -- llamada recursiva

El caso base es cuando el número es 0 o 1, ya que en esos casos, el número en la secuencia es igual al mismo número. La llamada recursiva se realiza sumando los dos números anteriores en la secuencia.

  2. Escribe una función recursiva para calcular el número de dígitos de un número entero.
  3. Escribe una función recursiva para encontrar el máximo elemento en una lista de números.
  4. Escribe una función recursiva para calcular el enésimo término en la secuencia de Fibonacci.
  5. Escribe una función recursiva que compruebe si una palabra es un palíndromo (se lee igual de izquierda a derecha y de derecha a izquierda).

Consejos y mejores prácticas¶

  • Siempre asegúrate de tener un caso base en tus funciones recursivas para evitar un bucle infinito.
  • Utiliza patrones de coincidencia para manejar diferentes casos en una función recursiva.
  • Asegúrate de que la llamada recursiva se acerca al caso base en cada iteración para evitar un stack overflow.
  • Utiliza funciones de orden superior para abstraer la recursión y hacer que tu código sea más legible y reusable.
  • Practica la recursión resolviendo diferentes problemas y retos para mejorar tus habilidades en esta técnica.
