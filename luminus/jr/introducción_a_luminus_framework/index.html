Introducción a Luminus Framework¶

Descripción del módulo¶

En este módulo se abordará el framework de desarrollo web Luminus, utilizado para crear aplicaciones en lenguaje Clojure. Se explicarán los conceptos básicos de su arquitectura y se realizarán ejemplos prácticos para comprender su funcionamiento.

Explicación teórica¶

Luminus es un framework de desarrollo web basado en Clojure, un lenguaje de programación funcional. Fue creado para simplificar la creación de aplicaciones web en Clojure, ya que proporciona una estructura predefinida y utilidades comunes para el desarrollo.

La arquitectura de Luminus se basa en el patrón de diseño Modelo-Vista-Controlador (MVC), donde el modelo representa los datos de la aplicación, la vista se encarga de mostrar la interfaz de usuario y el controlador maneja las interacciones entre el modelo y la vista. Además, Luminus utiliza el servidor web Jetty y el sistema de plantillas Hiccup para renderizar las vistas.

Palabras clave y su definición¶

  • Luminus: Framework de desarrollo web basado en Clojure.
  • Clojure: Lenguaje de programación funcional utilizado para crear aplicaciones en Luminus.
  • Arquitectura MVC: Patrón de diseño que divide una aplicación en tres componentes: modelo, vista y controlador.
  • Jetty: Servidor web utilizado por Luminus.
  • Hiccup: Sistema de plantillas para renderizar las vistas en Luminus.

Ejemplos de código en Luminus Framework¶

Crear una aplicación básica en Luminus¶

Primero, se debe instalar Leiningen, un gestor de dependencias para Clojure. Luego, ejecutar el siguiente comando en la terminal:

lein new luminus mi-app

Esto creará una carpeta llamada "mi-app" con la estructura básica de una aplicación en Luminus.

Definir una ruta y una vista en Luminus¶

En el archivo "mi-app/src/mi_app/handler.clj" se pueden definir las rutas de la aplicación. Por ejemplo:

(defroutes app-routes
  (GET "/" [] "¡Hola, mundo!")
  (GET "/saludo" [] (render "saludo.clj"))

En el archivo "mi-app/resources/templates/saludo.clj" se puede definir la vista correspondiente a la ruta "/saludo". Por ejemplo:

(html
  [:h1 "¡Bienvenido a Luminus!"]
  [:p "Esta es una vista de ejemplo."])

Ejercicios prácticos con instrucciones claras¶

  1. Crea una aplicación en Luminus y ejecútala en un servidor local.
  2. Agrega una nueva ruta a la aplicación y crea una vista correspondiente.
  3. Agrega una base de datos a la aplicación y realiza una consulta desde una de las rutas.
  4. Implementa un formulario en una de las vistas y guarda los datos en la base de datos.
  5. Crea una página de inicio de sesión y valida los datos ingresados con una base de datos de usuarios.

Consejos o mejores prácticas¶

  • Familiarizarse con el lenguaje Clojure antes de utilizar Luminus.
  • Seguir la convención de nombres de Luminus para la estructura de carpetas y archivos.
  • Utilizar las bibliotecas y utilidades recomendadas por Luminus para un desarrollo más eficiente.
  • Aprovechar la documentación y la comunidad de Luminus para resolver dudas y aprender nuevas técnicas.
