Internals de Phoenix¶

Explicación teórica¶

Phoenix es un framework de desarrollo web en Elixir que se basa en el patrón MVC (Modelo-Vista-Controlador) para estructurar aplicaciones web. Su arquitectura interna se basa en la filosofía de "convención sobre configuración", lo que significa que sigue una estructura predefinida para facilitar el desarrollo y la organización del código.

En términos generales, Phoenix consta de cuatro capas principales: el Router, el Controlador, el Modelo y la Vista. El Router es responsable de manejar las solicitudes HTTP y dirigirlas a los controladores adecuados. El Controlador se encarga de procesar las solicitudes y enviar una respuesta al cliente. El Modelo maneja la lógica de negocio y la interacción con la base de datos. Y la Vista se encarga de generar la interfaz de usuario y enviarla al cliente.

Palabras clave y su definición¶

  1. Phoenix: Framework de desarrollo web en Elixir.
  2. MVC: Patrón de diseño que divide una aplicación en tres capas: Modelo, Vista y Controlador.
  3. Convención sobre configuración: Filosofía de desarrollo que se basa en seguir una estructura predefinida en lugar de configurar cada parte de la aplicación.
  4. Router: Capa encargada de dirigir las solicitudes HTTP a los controladores adecuados.
  5. Controlador: Capa encargada de procesar las solicitudes y enviar una respuesta al cliente.
  6. Modelo: Capa encargada de manejar la lógica de negocio y la interacción con la base de datos.
  7. Vista: Capa encargada de generar la interfaz de usuario y enviarla al cliente.

Ejemplos de código en Phoenix Framework¶

# Router
scope "/", HelloPhoenixWeb do
  pipe_through :browser

  get "/", PageController, :index
end

# Controlador
defmodule HelloPhoenixWeb.PageController do
  use HelloPhoenixWeb, :controller

  def index(conn, _params) do
    render(conn, "index.html")
  end
end

# Modelo
defmodule HelloPhoenixWeb.User do
  use Ecto.Schema

  schema "users" do
    field :name, :string
    field :email, :string
    field :age, :integer
  end
end

# Vista
defmodule HelloPhoenixWeb.PageView do
  use HelloPhoenixWeb, :view

  def render("index.html", %{users: users}) do
    <ul>
      <%= for user <- users do %>
        <li><%= user.name %></li>
      <% end %>
    </ul>
  end
end

Ejercicios prácticos con instrucciones claras¶

  1. Crear una nueva ruta en el Router que dirija a un nuevo controlador llamado UserController.
  2. En el controlador UserController, crear una acción llamada "new" que renderice una vista con un formulario para crear un nuevo usuario.
  3. Crear un modelo User con los campos "name", "email" y "age".
  4. En el controlador, crear una acción "create" que reciba los datos del formulario y cree un nuevo usuario en la base de datos.
  5. En la vista, mostrar una lista de todos los usuarios creados hasta el momento.
  6. Agregar validaciones al modelo User para asegurarse de que los campos no estén vacíos y que la edad sea mayor de 18 años.
  7. Agregar una nueva acción en el controlador para editar los datos de un usuario existente.
  8. En la vista, agregar un enlace para editar cada usuario que muestre el formulario de edición correspondiente.
  9. Agregar una acción para eliminar usuarios en el controlador y un botón en la vista para borrarlos.
  10. Investigar cómo agregar estilo CSS a la vista y aplicarlo para mejorar la apariencia de la aplicación.

Consejos o mejores prácticas¶

  1. Sigue la convención sobre configuración para mantener una estructura organizada en tu aplicación.
  2. Utiliza buenas prácticas de programación como separar la lógica de la vista y evitar la duplicación de código.
  3. Aprovecha las herramientas y módulos disponibles en la comunidad de Elixir y Phoenix para facilitar tu trabajo.
  4. Familiarízate con Ecto, el ORM (Mapeo Objeto-Relacional) utilizado en Phoenix, para manejar las operaciones con la base de datos de manera eficiente.
  5. Asegúrate de entender los conceptos básicos de Elixir, como las funciones, las listas y las tuplas, ya que son fundamentales en el desarrollo con Phoenix.
  6. Practica y experimenta con los diferentes módulos y funcionalidades de Phoenix para mejorar tus habilidades y conocimientos.
