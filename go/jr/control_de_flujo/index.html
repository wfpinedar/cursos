Control de flujo¶

El control de flujo en Go lang se refiere a la forma en que el programa toma decisiones y maneja el flujo de ejecución en función de esas decisiones. Esto es esencial para crear programas eficientes y lógicos que puedan realizar tareas específicas de manera efectiva.

Explicación teórica¶

En Go lang, el control de flujo se logra mediante el uso de condicionales, bucles y otras estructuras de control. Las condicionales permiten al programa elegir entre diferentes acciones o rutas de ejecución en función de una o varias condiciones. Los bucles, por otro lado, permiten que una acción se repita varias veces hasta que se cumpla una condición específica.

Las estructuras de control de flujo también incluyen declaraciones de control, que son instrucciones que permiten al programador controlar directamente el flujo de ejecución del programa. Estas incluyen la declaración "break" para salir de un bucle, la declaración "continue" para saltar una iteración de un bucle y la declaración "goto" para saltar a una sección específica del código.

Palabras clave y su definición¶

  1. Condicionales: Son estructuras de control que permiten al programa elegir entre diferentes opciones de ejecución en función de una o varias condiciones.
  2. Bucles: Son estructuras de control que permiten que una acción se repita varias veces hasta que se cumpla una condición específica.
  3. Declaraciones de control: Son instrucciones que permiten al programador controlar directamente el flujo de ejecución del programa.
  4. If-else: Es una estructura de control condicional que ejecuta un bloque de código si se cumple una condición y otro bloque de código si no se cumple.
  5. For: Es una estructura de control de bucle que permite repetir una acción un número específico de veces.
  6. Break: Es una declaración de control que permite salir de un bucle o una estructura de control de flujo.
  7. Continue: Es una declaración de control que permite saltar una iteración de un bucle y continuar con la siguiente.
  8. Goto: Es una declaración de control que permite saltar a una sección específica del código.

Ejemplos de código en Go lang¶

If-else¶

num := 10
if num > 0 {
    fmt.Println("El número es positivo")
} else if num < 0 {
    fmt.Println("El número es negativo")
} else {
    fmt.Println("El número es cero")
}

For¶

for i := 0; i < 5; i++ {
    fmt.Println(i)
}

Break¶

for i := 0; i < 10; i++ {
    if i == 5 {
        break
    }
    fmt.Println(i)
}

Continue¶

for i := 0; i < 10; i++ {
    if i == 5 {
        continue
    }
    fmt.Println(i)
}

Goto¶

num := 10
if num > 0 {
    goto positive
} else {
    goto negative
}

positive:
    fmt.Println("El número es positivo")
    goto end

negative:
    fmt.Println("El número es negativo")

end:
    fmt.Println("Fin del programa")

Ejercicios prácticos con instrucciones claras¶

  1. Crea un programa que imprima los números del 1 al 10 utilizando un bucle "for".
  2. Modifica el programa anterior para que solo imprima los números pares.
  3. Crea un programa que pida al usuario un número y determine si es positivo, negativo o cero utilizando una estructura de control "if-else".
  4. Crea un programa que pida al usuario una palabra y determine si es un palíndromo (se lee igual de izquierda a derecha que de derecha a izquierda) utilizando un bucle "for".
  5. Crea un programa que pida al usuario un número y calcule su factorial utilizando un bucle "for".

Consejos o mejores prácticas¶

  • Utiliza nombres descriptivos para tus variables y declaraciones de control para que sea más fácil entender el flujo de tu programa.
  • Asegúrate de tener en cuenta todas las posibles condiciones y situaciones al utilizar declaraciones de control para evitar errores o resultados inesperados.
  • Utiliza comentarios para explicar el propósito de tus declaraciones de control y facilitar la lectura y el mantenimiento del código.
  • Practica con ejercicios y retos para mejorar tus habilidades en el uso de condicionales, bucles y otras estructuras de control de flujo en Go lang.
