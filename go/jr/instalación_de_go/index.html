Instalación de Go¶

Descripción del módulo¶

En este módulo, aprenderás cómo instalar y configurar Go en diferentes sistemas operativos. Go es un lenguaje de programación de código abierto desarrollado por Google, diseñado para ser eficiente, sencillo y rápido. A través de este módulo, podrás entender cómo instalar Go en tu sistema y cómo configurar tu entorno de desarrollo para empezar a programar en este lenguaje.

Palabras clave y su definición¶

  • Lenguaje de programación: conjunto de reglas y símbolos utilizados para crear programas informáticos.
  • Código abierto: software cuyo código fuente es accesible y modificable por cualquier persona.
  • Eficiencia: capacidad de lograr un resultado con el mínimo de recursos.
  • Sencillo: fácil de entender y utilizar.
  • Entorno de desarrollo: conjunto de herramientas y configuraciones utilizadas para escribir y depurar código.
  • Programar: escribir código para crear un programa informático.

Ejemplos de código en Go¶

Hola mundo en Go¶

package main

import "fmt"

func main() {
    fmt.Println("Hola mundo!")
}

Declaración de variables en Go¶

package main

import "fmt"

func main() {
    // Declaración de una variable entera
    var edad int
    edad = 25

    // Declaración de una variable string
    var nombre string = "Juan"

    // Declaración de una variable booleana
    var esMayor bool = true

    fmt.Println("Mi nombre es", nombre, "y tengo", edad, "años.")
    fmt.Println("¿Soy mayor de edad?", esMayor)
}

Ejercicios prácticos¶

  1. Instala Go en tu sistema operativo siguiendo los pasos que se explican a continuación.
  2. Configura tu entorno de desarrollo para programar en Go.
  3. Escribe un programa que solicite al usuario su nombre y edad, y luego imprima un mensaje con esa información.

Instrucciones para instalar Go en diferentes sistemas operativos¶

Windows¶

  1. Descarga el instalador de Go desde la página oficial: https://golang.org/dl/
  2. Ejecuta el instalador y sigue las instrucciones.
  3. Una vez finalizada la instalación, verifica que se haya configurado correctamente ejecutando el comando go version en la terminal. Debería mostrar la versión de Go instalada.

macOS¶

  1. Descarga el archivo .pkg de Go desde la página oficial: https://golang.org/dl/
  2. Ejecuta el archivo descargado y sigue las instrucciones.
  3. Una vez finalizada la instalación, verifica que se haya configurado correctamente ejecutando el comando go version en la terminal. Debería mostrar la versión de Go instalada.

Linux¶

  1. Abre la terminal y ejecuta el siguiente comando para descargar el archivo de instalación de Go: wget https://golang.org/dl/go<version>.linux-amd64.tar.gz (reemplaza <version> por la versión más reciente disponible).
  2. Descomprime el archivo descargado con el comando tar -xvf go<version>.linux-amd64.tar.gz.
  3. Mueve la carpeta resultante a una ubicación adecuada en tu sistema con el comando sudo mv go /usr/local.
  4. Configura las variables de entorno ejecutando los siguientes comandos en la terminal:
    • export GOPATH=$HOME/go
    • export PATH=$PATH:/usr/local/go/bin:$GOPATH/bin
  5. Verifica que se haya configurado correctamente ejecutando el comando go version en la terminal. Debería mostrar la versión de Go instalada.

Consejos y mejores prácticas¶

  • Es importante verificar que la instalación y configuración de Go se hayan realizado correctamente antes de comenzar a programar.
  • Siempre utiliza la versión más reciente de Go para aprovechar las mejoras y correcciones de errores.
  • Asegúrate de tener suficiente espacio en tu sistema antes de instalar Go, ya que suele ocupar varios gigabytes.
  • Sigue las instrucciones de instalación específicas para tu sistema operativo para evitar errores.
  • Actualiza regularmente tu entorno de desarrollo y las herramientas que utilices para programar en Go.
