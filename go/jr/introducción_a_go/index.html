Introducción a Go¶

Descripción del módulo¶

En este módulo, aprenderemos sobre Go, un lenguaje de programación de código abierto creado por Google en el año 2009. Exploraremos su breve historia, características y su uso en la actualidad. Al final de este módulo, tendrás una comprensión básica de Go y estarás listo para comenzar a escribir tus propios programas en este lenguaje.

Explicación teórica¶

Go es un lenguaje de programación compilado y concurrente diseñado para construir aplicaciones escalables y eficientes. Fue creado por Robert Griesemer, Rob Pike y Ken Thompson en Google, inspirados en otros lenguajes como C, Pascal y Oberon.

Una de las principales características de Go es su enfoque en la simplicidad y legibilidad del código. Esto se logra mediante la eliminación de características complejas y redundantes presentes en otros lenguajes. Go también se destaca por su capacidad para manejar la concurrencia de manera eficiente, lo que lo hace ideal para aplicaciones web y de servidor.

Palabras clave y su definición¶

  • Lenguaje de programación: un conjunto de símbolos y reglas utilizados para crear programas de computadora.
  • Código abierto: un software cuyo código fuente está disponible para su uso, modificación y distribución por parte de cualquier persona.
  • Compilado: un programa que traduce el código escrito por un programador a un lenguaje que la computadora puede entender y ejecutar.
  • Concurrente: la capacidad de realizar varias tareas al mismo tiempo.
  • Simplicidad: la cualidad de ser fácil de entender y usar.
  • Legibilidad: la facilidad con la que el código puede ser leído y comprendido por otros programadores.

Ejemplos de código en Go lang¶

Hola mundo¶

package main

import "fmt"

func main() {
    fmt.Println("Hola mundo")
}

Función factorial¶

package main

import "fmt"

func factorial(n int) int {
    if n == 0 {
        return 1
    }
    return n * factorial(n-1)
}

func main() {
    fmt.Println(factorial(5)) // salida: 120
}

  1. Escribe un programa en Go que imprima los números del 1 al 10.

  3. Escribe un programa en Go que ordene un arreglo de números de manera ascendente utilizando el algoritmo de burbuja.

Consejos o mejores prácticas¶

  • Utiliza nombres descriptivos para tus variables y funciones.
  • Aprovecha la concurrencia de Go para crear aplicaciones eficientes y escalables.
  • Lee y sigue las convenciones de estilo de código de Go para mantener una base de código consistente y legible.
